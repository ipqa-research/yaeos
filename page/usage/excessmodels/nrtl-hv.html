NRTL-HV

The NRLT model presented by Huron and Vidal is a variation of the original non-random two-liquid model. This variation is inteded to be used combined with cubic Equations of State, using the Huron-Vidal mixing rule. The major difference with respect to the original NRTL model is that it includes the repulsive parameter of the cubic EoS in the expression for the excess Gibbs energy. This allows the model to be simplified to the classic Van der Waals 1-fluid quadratic mixing rules when giving special values to the parameters.
